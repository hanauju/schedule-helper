# Schedule Helper Requirements

## MVP Users

오늘 해야 할 일을 정리하고, 특정 프로그램이나 화면에 실제로 머문 집중 시간을 기록하고 싶은 개인 사용자.

## Functional Requirements

- 사용자는 오늘 할 일을 빠르게 추가할 수 있다.
- 사용자는 오늘 일정을 빠르게 추가할 수 있다.
- 사용자는 집중할 일과 목표 시간을 정해 집중 세션을 시작할 수 있다.
- 사용자는 뽀모도로 기본 집중 시간과 휴식 시간을 조정할 수 있다.
- 사용자는 현재 열린 프로그램 목록에서 집중 대상 화면을 선택할 수 있다.
- 앱은 지정한 프로그램이 화면 맨 앞에 있을 때만 집중 시간으로 계산한다.
- 앱은 다른 프로그램으로 벗어난 시간을 이탈 시간으로 기록한다.
- 앱은 사용자가 일정 시간 이상 입력하지 않으면 자리 비움으로 보고 집중 시간에서 제외한다.
- 사용자는 집중 세션을 일시정지, 재개, 완료할 수 있다.
- 앱은 집중 시간, 이탈 시간, 일시정지 시간, 집중 유지율을 표시한다.
- 사용자는 빠른 메모를 작성할 수 있다.
- 앱은 메모 작성 시각을 자동 저장한다.
- 앱은 진행 중인 집중 세션과 메모를 연결할 수 있다.
- 앱은 작은 창에서 남은 시간과 집중 상태를 위젯처럼 보여준다.
- 앱은 모든 데이터를 SQLite에 저장한다.

## Non-Functional Requirements

- 앱은 Windows 데스크톱에서 실행된다.
- 주요 기능은 인터넷 연결 없이 동작한다.
- 집중 타이머와 저장소는 UI 없이 테스트할 수 있다.
- 저장소는 앱 재실행 후에도 데이터를 유지한다.
- 사용 시간과 메모 기록은 로컬 장치에만 저장된다.

## First Version Decisions

- UI는 PySide6 데스크톱 앱으로 만든다.
- 기본 화면은 Focus Desk로 구성한다.
- 샘플 데이터 버튼은 노출하지 않는다.
- 자동 배치 버튼은 첫 화면에서 제거한다.
- 프로그램 사용 시간은 Windows foreground window 기준으로 측정한다.
