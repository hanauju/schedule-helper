# Schedule Helper Requirements

## MVP Users

일정이 자주 바뀌고, 할 일을 주간 단위로 현실적인 시간대에 배치하고 싶은 개인 사용자.

## Functional Requirements

- 사용자는 작업을 추가, 수정, 삭제할 수 있다.
- 작업은 이름, 예상 소요 시간, 마감일, 우선순위, 카테고리, 완료 상태를 가진다.
- 사용자는 고정 일정을 추가, 수정, 삭제할 수 있다.
- 사용자는 요일별 사용 가능 시간대를 설정할 수 있다.
- 사용자는 하루 최대 자동 배치 시간과 작업 사이 휴식 시간을 설정할 수 있다.
- 앱은 사용 가능 시간대 안에서 고정 일정과 충돌하지 않게 작업을 배치한다.
- 앱은 마감일이 가까운 작업과 우선순위가 높은 작업을 먼저 배치한다.
- 앱은 배치하지 못한 작업과 이유를 사용자에게 보여준다.
- 앱은 모든 데이터를 SQLite에 저장한다.

## Non-Functional Requirements

- 앱은 Windows 데스크톱에서 실행된다.
- 주요 기능은 인터넷 연결 없이 동작한다.
- 일정 생성 로직은 UI 없이 테스트할 수 있다.
- 저장소는 앱 재실행 후에도 데이터를 유지한다.

## First Version Decisions

- UI는 PySide6 데스크톱 앱으로 만든다.
- 기본 화면은 주간 캘린더 중심으로 구성한다.
- 자동 배치 기준은 마감일 우선, 우선순위 보조 기준으로 둔다.
- 외부 캘린더 연동은 첫 버전에서 제외한다.
