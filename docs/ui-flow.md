# Schedule Helper UI Flow

## Main Layout

- Left: 작업 입력 폼과 작업 목록
- Center: 주간 캘린더
- Right: 일정 편집 탭과 사용 가능 시간 설정 탭
- Toolbar: 이전 주, 오늘, 다음 주, 자동 배치, 샘플 데이터

## Core Flow

1. 사용자가 작업을 입력한다.
2. 사용자가 고정 일정을 입력한다.
3. 사용자가 사용 가능 시간대를 확인하거나 수정한다.
4. 사용자가 자동 배치를 실행한다.
5. 앱이 주간 캘린더에 생성된 일정을 표시한다.
6. 사용자가 필요한 일정을 클릭해 직접 수정한다.

## Manual Event Flow

1. 캘린더 빈 칸을 클릭한다.
2. 우측 일정 폼에 선택한 시간이 들어간다.
3. 제목과 종료 시간을 조정한다.
4. 일정 저장을 누른다.
5. 충돌이 있으면 경고를 확인한다.

## Task Flow

1. 좌측 작업 폼에 이름, 소요 시간, 마감, 우선순위를 입력한다.
2. 작업 저장을 누른다.
3. 목록에서 작업을 선택하면 폼에서 다시 수정할 수 있다.
4. 완료 전환으로 작업 상태를 바꿀 수 있다.
