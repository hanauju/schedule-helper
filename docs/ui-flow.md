# Schedule Helper UI Flow

## Main Layout

- Top: 오늘 날짜와 위젯 모드 전환
- Center: Focus Desk
- Lower Left: 오늘 흐름
- Lower Right: 빠른 메모
- Compact Mode: 남은 시간, 집중 상태, 일시정지/완료, 빠른 메모

## Focus Flow

1. 사용자가 집중할 일을 입력한다.
2. 현재 열린 프로그램 목록에서 집중 대상 화면을 선택한다.
3. 목표 시간, 뽀모도로 시간, 휴식 시간, 자리 비움 기준을 확인한다.
4. 시작을 누른다.
5. 앱은 1초마다 현재 foreground window를 확인한다.
6. 대상 프로그램이면 집중 시간으로 기록한다.
7. 대상 프로그램이 아니면 이탈 시간으로 기록한다.
8. 사용자는 일시정지, 재개, 완료를 누를 수 있다.

## Today Flow

1. 사용자가 오늘 할 일을 빠르게 추가한다.
2. 사용자가 오늘 일정을 빠르게 추가한다.
3. 오늘 흐름 목록에서 할 일을 선택한다.
4. 선택 집중을 누르면 Focus Desk에 해당 일이 들어간다.
5. 완료 처리를 누르면 할 일이 완료된다.

## Quick Memo Flow

1. 사용자가 빠른 메모 입력창에 내용을 적는다.
2. `Ctrl+Enter` 또는 저장 버튼으로 메모를 저장한다.
3. 앱은 작성 시각을 자동 저장한다.
4. 진행 중인 집중 세션이 있으면 메모가 해당 세션에 연결된다.
5. 최근 메모는 최신순으로 표시된다.

## Compact Flow

1. 사용자가 위젯 모드를 누르거나 창을 작게 줄인다.
2. 앱은 compact layout으로 전환된다.
3. 남은 시간과 집중 유지 상태가 크게 표시된다.
4. 사용자는 일시정지/재개, 완료, 빠른 메모 저장을 할 수 있다.
5. 전체 버튼으로 Focus Desk 화면으로 돌아간다.
