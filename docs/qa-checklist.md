# QA Checklist

## App Launch

- [ ] 앱이 오류 없이 실행된다.
- [ ] 첫 화면이 Focus Desk 중심으로 열린다.
- [ ] 샘플 데이터 버튼이 보이지 않는다.
- [ ] 자동 배치 버튼이 보이지 않는다.

## Windows Chrome

- [ ] Windows 작업 표시줄 자동 숨기기를 켠 상태에서 앱을 최대화하고, 화면 하단 끝으로 커서를 이동하면 작업 표시줄이 나타난다.

## Focus Session

- [ ] 집중할 일을 입력할 수 있다.
- [ ] 현재 열린 프로그램 목록이 드롭다운에 표시된다.
- [ ] 목표 시간을 설정할 수 있다.
- [ ] 시작을 누르면 타이머가 시작된다.
- [ ] 일시정지 중에는 집중 시간이 증가하지 않는다.
- [ ] 재개하면 집중 시간이 다시 증가한다.
- [ ] 완료하면 집중 기록이 저장된다.
- [ ] 대상 프로그램을 벗어나면 이탈 시간이 증가한다.
- [ ] 자리 비움 기준을 넘으면 집중 시간에서 제외된다.

## Today Flow

- [ ] 오늘 할 일을 빠르게 추가할 수 있다.
- [ ] 오늘 일정을 빠르게 추가할 수 있다.
- [ ] 오늘 흐름에서 할 일을 선택해 Focus Desk로 보낼 수 있다.
- [ ] 선택한 할 일을 완료 처리할 수 있다.

## Quick Memo

- [ ] 빠른 메모를 저장할 수 있다.
- [ ] `Ctrl+Enter`로 메모가 저장된다.
- [ ] 메모 작성 시각이 표시된다.
- [ ] 진행 중인 세션이 있으면 메모가 세션과 연결된다.
- [ ] compact mode에서도 메모를 저장할 수 있다.

## Compact Mode

- [ ] 위젯 모드 버튼으로 compact mode에 들어갈 수 있다.
- [ ] 창을 작게 줄이면 compact mode로 전환된다.
- [ ] compact mode에서 남은 시간이 보인다.
- [ ] compact mode에서 일시정지/재개가 가능하다.
- [ ] 전체 버튼으로 원래 화면으로 돌아갈 수 있다.

## Persistence

- [ ] 앱을 재실행해도 작업이 유지된다.
- [ ] 앱을 재실행해도 일정이 유지된다.
- [ ] 앱을 재실행해도 집중 기록이 유지된다.
- [ ] 앱을 재실행해도 메모가 유지된다.

## 워크스페이스 전환 및 유지

- [ ] 상단 `작업공간` 버튼으로 워크스페이스 목록을 열 수 있다.
- [ ] 다른 워크스페이스를 선택하면 화면의 메모와 체크리스트가 즉시 전환된다.
- [ ] 워크스페이스 전환 후 앱을 재실행하면 마지막 선택이 유지된다.
- [ ] 워크스페이스 관리 다이얼로그에서 워크스페이스를 만들고 이름을 변경할 수 있다.
- [ ] 워크스페이스 관리 다이얼로그에서 워크스페이스를 삭제할 때 남은 워크스페이스가 정상 선택된다.

## 메모 정렬/고정/태그/전체 스크롤

- [ ] 메모 정렬 아이콘 버튼으로 최신순과 오래된순을 전환할 수 있다.
- [ ] 고정한 메모가 정렬 순서와 관계없이 상단에 표시된다.
- [ ] 메모에 할당된 태그가 배지로 표시된다.
- [ ] 긴 메모 목록에서 전체 스크롤이 끊기지 않고 마지막 메모까지 이동한다.
- [ ] 메모 컨텍스트 메뉴에서 고정, 태그 할당, 삭제 동작을 실행할 수 있다.

## 체크리스트 정렬/고정/태그/필터

- [ ] 체크리스트 정렬 아이콘 버튼으로 최신순과 오래된순을 전환할 수 있다.
- [ ] 고정한 체크리스트가 정렬 순서와 관계없이 상단에 표시된다.
- [ ] 체크리스트에 할당된 태그가 배지로 표시된다.
- [ ] 체크리스트 필터로 전체, 진행 중, 완료 항목을 전환할 수 있다.
- [ ] 체크리스트 컨텍스트 메뉴에서 고정, 태그 할당, 삭제 동작을 실행할 수 있다.

## 태그 관리

- [ ] 태그 관리 화면에서 새 태그를 생성할 수 있다.
- [ ] 기존 태그 이름을 변경하면 연결된 메모와 체크리스트 배지가 함께 갱신된다.
- [ ] 태그를 삭제하면 연결된 메모와 체크리스트에서 해당 태그가 제거된다.
- [ ] 메모와 체크리스트에 태그를 할당하고 해제할 수 있다.

## 기존 기능 회귀

- [ ] 집중 세션 시작, 일시정지, 재개, 완료, 이탈 시간 계산이 기존과 동일하게 동작한다.
- [ ] Today 흐름에서 할 일과 일정을 추가하고 Focus Desk로 보내며 완료 처리할 수 있다.
- [ ] compact mode 진입, 남은 시간 표시, 일시정지/재개, 전체 화면 복귀가 동작한다.
- [ ] 앱 재실행 후 작업, 일정, 집중 기록, 메모, 워크스페이스 선택, 태그 할당이 유지된다.
