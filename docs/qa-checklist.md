# QA Checklist

## App Launch

- [ ] 앱이 오류 없이 실행된다.
- [ ] 첫 화면이 Focus Desk 중심으로 열린다.
- [ ] 샘플 데이터 버튼이 보이지 않는다.
- [ ] 자동 배치 버튼이 보이지 않는다.

## Focus Session

- [ ] 집중할 일을 입력할 수 있다.
- [ ] 현재 열린 프로그램 목록이 드롭다운에 표시된다.
- [ ] 목표 시간을 설정할 수 있다.
- [ ] 시작을 누르면 타이머가 시작된다.
- [ ] 일시정지 중에는 집중 시간이 증가하지 않는다.
- [ ] 재개하면 집중 시간이 다시 증가한다.
- [ ] 완료하면 집중 기록이 저장된다.
- [ ] 대상 프로그램을 벗어나면 이탈 시간이 증가한다.
- [ ] 자리 비움 기준을 넘으면 집중 시간에서 제외된다.

## Today Flow

- [ ] 오늘 할 일을 빠르게 추가할 수 있다.
- [ ] 오늘 일정을 빠르게 추가할 수 있다.
- [ ] 오늘 흐름에서 할 일을 선택해 Focus Desk로 보낼 수 있다.
- [ ] 선택한 할 일을 완료 처리할 수 있다.

## Quick Memo

- [ ] 빠른 메모를 저장할 수 있다.
- [ ] `Ctrl+Enter`로 메모가 저장된다.
- [ ] 메모 작성 시각이 표시된다.
- [ ] 진행 중인 세션이 있으면 메모가 세션과 연결된다.
- [ ] compact mode에서도 메모를 저장할 수 있다.

## Compact Mode

- [ ] 위젯 모드 버튼으로 compact mode에 들어갈 수 있다.
- [ ] 창을 작게 줄이면 compact mode로 전환된다.
- [ ] compact mode에서 남은 시간이 보인다.
- [ ] compact mode에서 일시정지/재개가 가능하다.
- [ ] 전체 버튼으로 원래 화면으로 돌아갈 수 있다.

## Persistence

- [ ] 앱을 재실행해도 작업이 유지된다.
- [ ] 앱을 재실행해도 일정이 유지된다.
- [ ] 앱을 재실행해도 집중 기록이 유지된다.
- [ ] 앱을 재실행해도 메모가 유지된다.
